# Simple Clicker Game for Android

Это простой проект игры-кликера для Android.

## Функционал:
- **Основной клик** - большая зелёная кнопка для набора очков
- **Улучшение клика** - увеличивает количество очков за один клик
- **Авто-кликер** - автоматически добавляет очки каждую секунду

## Как запустить:

### Вариант 1: Android Studio (рекомендуется)
1. Откройте Android Studio
2. Выберите `File` → `Open`
3. Выберите папку `/workspace/ClickerGame`
4. Дождитесь завершения синхронизации Gradle
5. Запустите на эмуляторе или реальном устройстве

### Вариант 2: Командная строка
```bash
cd /workspace/ClickerGame
./gradlew assembleDebug
```
APK файл будет создан в `app/build/outputs/apk/debug/`

- Целевая версия Android: 14 (API 34)
- Язык: Java
